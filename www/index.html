How it works

Perceive-act loop

Tell your agent what you want. It drives the browser.

01
Act
Navigate, click, type, scroll - 50+ CDP commands. Each one auto-prints a compact page brief.
02
Perceive
Read the brief (~200 chars). Need more? Use dom (~4k), axtree (~500), or screenshot as fallback.
03
Decide
The agent adapts. Continue, try another approach, or report results. Repeat until the goal is done.
Architecture

Direct line to the browser

No daemon, no framework, no abstraction layers. Every Playwright-based tool adds hops between the agent and the browser.

Get started

One command. No setup.

Works with Claude Code, Cursor, Codex, Windsurf, Cline, Copilot, OpenCode, Goose, and more. No npm install, no browser download.

All agents

# Cursor, Codex, Windsurf, Cline, Copilot, OpenCode, Goose, and more
npx skills add kxbnb/webact

Claude Code

# inside Claude Code
/plugin marketplace add kxbnb/webact
/plugin install webact@webact

Manual

# copy into your project
git clone https://github.com/kxbnb/webact.git /tmp/webact
cp -r /tmp/webact/skills/webact .agents/skills/webact
